XSLT 2.0 is simply the new iteration of XSLT 1.0 that adds many useful features and enhancements. Few of those enhancements include schema awareness , enhanced typing, enhanced built in functions (which is my favorite enhancement) , enhanced XSLT instruction set, regular expression support and other enhancements.

XPATH 2.0 is again the later recommendation from W3C. XPATH 2.0 supports data types based on XML schemas and has many powerful built in functions, operators and constructs. XPATH 2.0 has greater flexibility in what it can return which could include simple-typed values, nodes, sequences of nodes or simple-typed values. And finally XPATH 2.0 has a new pragmatic way of treating everything as a sequence
